My first actualized video game design project for a game design class, MazeGrappler is a first person maze game that takes inspriation from pacman and Ristar. it was very rushed as an experience, and I made some mistakes that I regret making, but it taught me alot about game design. Many aspects of Mazegrappler are found and implemented in my latest project, Project Railgun, notably my usage of multi tier level design and sections that homage the maze design.
